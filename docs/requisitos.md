# Requisitos

## Requisitos Funcionais

### RF01
O sistema deverá exibir os produtos da Criffi.

### RF02
O usuário poderá visualizar informações de cada produto.

### RF03
O usuário poderá filtrar produtos por categoria.

### RF04
O usuário poderá navegar pelas seções utilizando a navbar.

### RF05
O usuário poderá entrar em contato pelo WhatsApp.

### RF06
O sistema deverá possuir scroll suave entre as seções.

---

## Requisitos Não Funcionais

### RNF01
O site deverá ser responsivo.

### RNF02
O carregamento deverá ser rápido.

### RNF03
Compatibilidade com Chrome, Edge e Firefox.

### RNF04
Código organizado e documentado.

### RNF05
Utilizar apenas HTML, CSS e JavaScript.

---

## Regras de Negócio

- Os produtos serão apenas exibidos.
- Não haverá compra online.
- O atendimento será realizado pelo WhatsApp.
